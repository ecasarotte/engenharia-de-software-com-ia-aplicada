**O que é um tensor?**

No TensorFlow.js, um **tensor** é simplesmente um array numérico – pense nisso como uma lista (ou uma matriz) que armazena apenas números.
Diferente de um objeto JavaScript comum (como `{nome: "Erick", idade: 30, cor: "azul"}`), um tensor não contém nomes de propriedades, tipos de dados ou valores não numéricos. Ele é uma estrutura limpa e homogênea que os algoritmos de machine learning podem processar facilmente.

**Por que precisamos de tensores?**

O TensorFlow.js não entende diretamente objetos JavaScript ou valores primitivos. Antes que qualquer algoritmo possa aprender com os dados, você deve **converter** os dados brutos em um tensor – uma estrutura que contenha apenas números.

**Representação no JavaScript**

No JavaScript, um tensor pode ser escrito de algumas maneiras simples:

```js
// 1️⃣ Tensor unidimensional (vetor)
const vetor1D = [1, 2, 3];

// 2️⃣ Tensor bidimensional (matriz)
const matriz2D = [
  [1, 0, 0],   // Erick – cor azul (um-hot para azul)
  [0, 1, 0],   // Ana   – cor vermelha (um-hot para vermelho)
  [0, 0, 1]    // Carlos– cor verde (um-hot para verde)
];
```

No exemplo que vimos, queríamos capturar a **cor preferida** de cada pessoa. Como temos três cores possíveis (`azul`, `vermelho`, `verde`), criamos uma matriz 3 × 3 onde cada linha é um registro e cada coluna representa uma cor com codificação one-hot:

| Pessoa | Azul | Vermelho | Verde |
|--------|------|--------|-------|
| Erick  | 1    | 0      | 0     |
| Ana    | 0    | 1      | 0     |
| Carlos | 0    | 0      | 1     |

Essa matriz é exatamente um **tensor 2D** em JavaScript. Cada entrada é um número (0 ou 1), e toda a estrutura pode ser passada diretamente para um modelo do TensorFlow.js.

**Um fluxo de trabalho rápido e prático**

1. **Comece com dados brutos** (arrays de objetos).
   ```js
   const pessoas = [
     { nome: "Erick", idade: 30, cor: "azul", localizacao: "São Paulo" },
     { nome: "Ana",   idade: 25, cor: "vermelho", localizacao: "Rio" },
     { nome: "Carlos", idade: 40, cor: "verde", localizacao: "Curitiba" }
   ];
   ```

2. **Extraia apenas os números** que você precisa (neste caso, a cor).
   ```js
   const cores = ["azul", "vermelho", "verde"];
   const tensor = pessoas.map(p => {
     return cores.map(c => (p.cor === c ? 1 : 0));
   });
   // → [[1,0,0], [0,1,0], [0,0,1]]
   ```

3. **Use o tensor** em qualquer modelo do TensorFlow.js.
   ```js
   const modelo = tf.sequential();
   modelo.add(tf.layers.dense({inputShape: [3], units: 1}));
   // … treine e faça previsões com `tensor`
   ```

**Resumo**

- Um **tensor** é um array numérico (unidimensional, bidimensional, etc.).
- No JavaScript, você pode escrevê-lo como um array normal (`[1,2,3]`) ou como uma matriz aninhada (`[[1,0,0], [0,1,0], [0,0,1]]`).
- A conversão de objetos do mundo real em tensores envolve a extração de valores numéricos e a transformação deles em um formato consistente (por exemplo, codificação one-hot).

Entender tensores é o primeiro passo para trabalhar com qualquer modelo de machine learning no TensorFlow.js, pois é a única forma de representar dados que o motor de ML pode processar. 🚀
